# Star Chef - AI Menu Engine v0.3

Static review prototype for the Huazhu project menu engine.

This build is for internal business review only. It demonstrates:

- meal-period and stall opening rules
- procurement deduplication by dish pool
- chef preparation task generation
- target cost ceiling and margin-risk screening
- explicit data-quality gates before true gross-margin calculation

The original Excel workbooks and source screenshots are not included in this public demo repository.
